# Legacy Content (roomfit.kr)

> **출처**: 기존 roomfit.kr 웹사이트에서 스크랩한 콘텐츠
> **용도**: 참고용 아카이브 (신규 랜딩페이지 개발 시 레퍼런스)
> **상태**: 아카이브 (수정하지 않음)

## 구조

```
legacy/
├── content/
│   ├── raw-text.txt           # 원본 텍스트 (스크랩)
│   └── roomfit-content.json   # 구조화된 JSON
└── images/                    # 75개 이미지 파일
```

## 참고사항

- 이 콘텐츠는 **신규 랜딩페이지 개발의 레퍼런스**로만 사용
- 실제 적용 콘텐츠는 `docs/master/` 마스터 문서 참조
- 이미지 중 필요한 것은 `public/roomfit/`으로 복사하여 사용
