# 홈트 일반인 타겟 소개 문구

## 1. AI PT (VBT 분석)

### 트레이너 없이도 정확한 운동, AI가 당신의 모든 동작을 분석합니다

혼자 운동할 때 가장 어려운 건 '제대로 하고 있는지' 확인하는 것입니다. Roomfit은 VBT(속도 기반 트레이닝) 기술로 당신의 모든 동작을 실시간 분석합니다. 세는 데 신경 쓸 필요 없이 운동에만 집중하세요. 대충 한 동작은 세지 않아 스스로 정확한 운동 습관이 형성됩니다.

**핵심 가치:**
- 내가 얼마나 폭발적으로 들어올리는지 **속도(m/s)**로 확인
- "오늘 파워 신기록!" **파워(W)** 측정으로 새로운 성취 지표
- 트레이너 없이도 내 동작이 일정한지 **품질 점수**로 확인
- "지금 피로 누적됨" 알림으로 부상 위험 전 운동 중단 유도

**주요 기능:**
- 자동 렙 카운팅 (유효 동작만 카운트)
- 실시간 속도/파워 시각화
- 동작 품질(Quality) 점수 평가
- 피로도 자동 감지 & RIR(남은 여력) 예측
- 1RM 추정 (위험한 테스트 없이 최대 근력 파악)

---

## 2. 핸즈프리 간편 조작

### 손을 놓지 않고, 말 한마디로 무게를 조절하세요

고중량 운동 중 스마트폰을 조작하려고 자세를 풀지 마세요. "웨이트 온" 한마디면 충분합니다. 덤벨 내려놓고 핀 빼고 할 필요 없이, 음성 명령과 제스처만으로 운동 흐름을 유지할 수 있습니다.

**핵심 가치:**
- 양손이 자유로운 상태에서 안전하고 효율적인 운동
- 명령이 제대로 들어갔는지 바로 확인 가능
- 그립 놓치면 자동 무게 OFF로 안전 확보

**주요 기능:**
- 음성 명령: "웨이트 온"
- 실시간 명령 인식 피드백 UI
- **제스처 조작 (신규)**
  - Drop 자동 무게 Off (그립 놓칠 시)
  - 휴식 자동 무게 Off (가만히 있을 시)
  - 빠르게 당겨서 무게 On

## 3. 스마트 웨이트 모드 & 드랍세트

### 혼자서도 헬스장 고급 기법, 드랍세트가 가능합니다

핀 교체 없이 터치 한 번으로 무게 설정을 끝내세요. 특히 드랍세트 설정 시, 플레이트 교체 없이 쉬지 않고 연속 수행이 가능합니다. 내 무게 수준에 맞게 비율(%)로 자동 계산되어 복잡한 설정이 필요 없습니다.

**지원 모드:**
- **Constant:** 일반 헬스장 기구처럼 익숙한 느낌으로 운동
- **Negative:** 내리는 동작에서 더 강한 자극 → 근비대 효과 극대화
- **Band:** 관절에 부담 적은 초반 + 강한 마무리로 안전한 운동

**드랍세트:**
- "5kg씩 줄여가며 3단계" 단순하게 설정
- 내 무게 수준에 맞게 비율(%)로 자동 계산
- "2단계" 시각적 동기부여로 완주 유도

## 4. 개인화된 ROM (가동범위) 최적화

### 따로 설정 안 해도 내 가동범위에 맞춰 자동 세팅

사람마다 팔다리 길이와 유연성이 다릅니다. Roomfit은 몇 번 움직이면 최적의 가동범위를 자동으로 설정합니다. 부상 부위가 있다면 제한된 범위만 운동하도록 수동 설정도 가능합니다.

**주요 기능:**
- 자동 ROM 캘리브레이션
- 부상 부위 피해 제한 범위 설정
- 수동 구간 설정 (Top/Bottom, Left/Right)
- 정적/동적 측정 모드

---

## 5. 기록 & 성장 분석

### 운동 잔디를 채우는 재미, GitHub처럼 쌓이는 운동 기록

운동만 하면 자동 기록됩니다. 귀찮게 수기 입력할 필요 없이, 매일의 운동 기록이 쌓여 6개월간의 '활동 히트맵(잔디)'을 완성합니다. "7일 연속!" 스트릭이 끊기 싫어서 오늘도 운동하게 됩니다.

**핵심 가치:**
- 운동한 날 표시 보며 뿌듯함 느끼고 동기부여
- "한 달 전보다 확실히 강해졌네!" 성장 실감
- "저번 주보다 5kg 늘었네!" 성장 데이터로 확인

**주요 기능:**
- 캘린더 뷰 & 운동 강도별 히트맵 시각화
- 연속 운동 스트릭(Streak) & 유지 알림
- 세션/세트/렙 단위 상세 데이터 로그
- 퍼포먼스 변화 트렌드 분석

## 6. 소셜 공유 & 데이터 내보내기

### 인스타에 오운완 인증샷, 예쁜 브랜딩 이미지로 자랑하세요

오늘 달성한 운동 성과를 Roomfit 전용 브랜딩 이미지로 생성하여 SNS에 즉시 공유하세요. "나 이번 달 개근했어!" 히트맵으로 자랑할 수 있습니다. 더 깊은 분석이 필요하면 VBT 데이터를 CSV로 추출하세요.

**주요 기능:**
- SNS 공유용 이미지 생성 (운동 요약/히트맵)
- 저장 후 앱 전환 없이 바로 SNS 공유
- VBT Raw 데이터 CSV 내보내기

## 7. 디바이스 연결

### 앱 켜면 알아서 기기 찾아줌, 복잡한 설정 없이 바로 운동

복잡한 블루투스 설정 없이 탭 한 번으로 연결됩니다. 연결 끊겨도 당황 안 하고 자동 복구됩니다. 새 기능이 계속 추가되어 기기 가치가 상승합니다.

**주요 기능:**
- 자동 스캔 & 원터치 페어링
- 자동 재연결
- OTA 펌웨어 업데이트

# PT샵/헬스장 타겟 소개 문구

## 1. AI PT (VBT 분석)

### 트레이너가 카운팅 대신 자세 교정과 동기부여에 집중할 수 있습니다

회원의 운동을 AI가 정확히 측정하고 기록합니다. 트레이너는 카운팅에서 벗어나 진짜 중요한 자세 교정과 동기부여에 집중하세요. 회원이 치팅 없이 정확히 수행했는지 데이터로 확인 가능합니다.

**핵심 가치:**
- 선수/고급 회원에게 **속도 기반 훈련 프로그램** 설계
- 운동선수 회원에게 **파워 향상 추이** 보고 및 훈련 조정
- 회원 동작 품질을 **수치화하여 개선 포인트** 명확히 제시
- 회원 **과훈련 방지**, 적정 볼륨에서 세트 종료 권고

**전문 코칭 기능:**
- 회원 1RM을 안전하게 추정하여 **%1RM 기반 프로그램 설계**
- 회원에게 최적의 **Target Rep**을 전문 데이터와 함께 코칭
- 훈련 목표별(근력/근비대) **최적 속도 손실률**로 세트 관리
- 회원 목표에 맞는 **속도 존** 설정 및 모니터링

**차별화 포인트:**
> 속도 측정, 파워 계산, 가속도 분석은 **전문 장비 또는 유사품에서만 가능**한 룸핏만의 독보적 기능입니다.

## 2. 핸즈프리 간편 조작

### 회원 자세 잡아주면서 동시에 무게 조절이 가능합니다

트레이너가 회원의 자세를 교정하면서 동시에 무게를 조절할 수 있습니다. 시끄러운 환경에서도 명령 인식 여부를 즉시 파악하고, 수업 흐름이 끊기지 않게 빠른 문제 해결이 가능합니다.

**핵심 가치:**
- 여러 회원 관리 중에도 기기 상태 한눈에 파악
- 수업 흐름 끊기지 않게 빠른 문제 해결
- 회원 안전을 위한 자동 무게 OFF 기능

**주요 기능:**
- 음성 명령으로 핸즈프리 무게 조절
- 실시간 명령 인식 피드백 UI
- **제스처 조작 (신규)**
  - Drop 자동 무게 Off
  - 휴식 자동 무게 Off
  - 빠르게 당겨서 무게 On

---

## 3. 스마트 웨이트 모드 & 드랍세트

### 시간 효율적인 고강도 수업 구성으로 PT를 차별화하세요

트레이너가 무게 바꾸는 동안 회원이 쉬는 시간을 제로로 만듭니다. 정체기 회원에게 새로운 자극을 제공하고, 재활/초보 회원에게 관절 부담을 줄인 점진적 저항을 제공할 수 있습니다.

**지원 모드:**
- **Constant:** 기본 웨이트 트레이닝 수업에 활용
- **Negative:** 정체기 회원에게 새로운 자극 제공, 고급 테크닉 코칭
- **Band:** 재활/초보 회원에게 관절 부담 줄인 점진적 저항 제공

**드랍세트:**
- 초보 회원에게 이해하기 쉬운 드랍세트 설계 (절대값 감소)
- 회원별 무게에 상관없이 일관된 드랍 비율 적용 (퍼센트 감소)
- 회원 레벨에 맞춘 단계별 볼륨 정밀 조절
- 회원에게 "한 단계만 더!" 독려하며 완주 유도

---

## 4. 개인화된 ROM (가동범위) 최적화

### 회원 교체 시 자동으로 각자 ROM이 적용됩니다

회원별 최적 ROM 데이터가 축적되어 회원 교체 시 자동으로 각자의 가동범위가 적용됩니다. 재활 회원이나 특수 목적 훈련 시 정밀 ROM 설정이 가능하고, 좌우 비대칭 교정 훈련 시 측별 ROM을 다르게 설정할 수 있습니다.

**주요 기능:**
- 회원별 최적 ROM 데이터 축적 및 활용
- 재활 회원 / 특수 목적 훈련 시 정밀 ROM 설정
- 좌우 비대칭 교정 훈련 시 측별 ROM 다르게 설정
- 회원의 실제 운동 패턴 기반 ROM 설정

## 5. 회원 관리 & 성과 분석

### 회원에게 장기 성장 그래프 보여주며 PT 효과를 입증하세요

회원별 모든 운동 이력이 자동 누적되어 관리가 편리합니다. 회원 진도를 파악하고 다음 수업 계획을 수립하세요. "한 달 전보다 이만큼 늘었어요" 데이터로 PT 성과를 입증할 수 있습니다.

**핵심 가치:**
- 회원 출석/운동 빈도 한눈에 파악
- 회원별 세트 간 피로 패턴 분석하여 맞춤 세트 수 설계
- 회원에게 세트별 피드백 제공 → **전문성 어필**
- 회원에게 장기 성장 그래프 보여주며 **PT 효과 입증**

**주요 기능:**
- 회원별 운동 이력 자동 누적
- 회원 특정일 운동 내용 빠른 조회
- 세션/세트/렙 단위 상세 데이터 로그
- 퍼포먼스 변화 트렌드 분석
- 회원에게 보낼 일일 피드백 자동 생성

---

## 6. 소셜 공유 & 데이터 내보내기

### 센터 SNS에 회원 성과 홍보, 룸핏 사용 센터임을 자연스럽게 어필

회원 성과를 Roomfit 브랜딩 이미지로 생성하여 센터 SNS에 홍보하세요 (동의 하에). 회원 데이터를 외부 분석 도구로 심층 분석하거나, 연구 목적으로 활용할 수 있습니다.

**주요 기능:**
- 센터 SNS 마케팅용 브랜딩 이미지 생성
- 빠른 SNS 마케팅 콘텐츠 생성
- 회원 데이터 보고서 작성
- VBT Raw 데이터 CSV 내보내기 (연구/분석용)

## 7. 디바이스 연결 & 관리

### 여러 대 기기 중 빠르게 원하는 기기 연결, 일괄 업데이트 가능

회원 교체 시 빠른 기기 전환이 가능합니다. 수업 중 연결 끊김으로 인한 흐름 방해를 최소화하고, 여러 대 기기 일괄 업데이트로 관리 효율을 높입니다. 영업 중 기기 장애 리스크를 최소화합니다.

**주요 기능:**
- 여러 대 기기 중 빠르게 원하는 기기 연결
- 여러 기기 연결 상태 동시 모니터링
- 수업 전 기기 배터리 상태 점검
- 여러 대 기기 일괄 OTA 업데이트
- 업데이트 실패 시 롤백 복구 지원

## 8. 빠르고 직관적인 운동 시작

### 회원 교체 시 복잡한 세팅 없이 바로 다음 수업 시작

단골 회원 세팅값을 바로 불러와서 시간을 절약하세요. 인기 운동이나 회원별 자주 하는 운동에 빠르게 접근할 수 있습니다. 회원 목표에 맞는 다양한 운동 옵션을 제안하세요.

**주요 기능:**
- 리모컨 모드: 복잡한 설정 없이 바로 운동 시작
- 이전 설정 불러오기: 단골 회원 세팅값 원터치 복원
- 즐겨찾기 운동: 인기 운동 / 회원별 자주 하는 운동 빠른 접근
- 운동 라이브러리: 부위별/이름별 운동 검색
- 그립 타입 변형: 회원 손목/어깨 상태에 맞는 그립 선택하여 부상 방지

## 룸핏만의 독보적 기능 (효용성 5점)
