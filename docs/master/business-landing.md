# 비즈니스 랜딩 페이지 마스터 문서

> **페이지**: 비즈니스 (B2B - 트레이너/센터)
> **URL**: `/business`
> **목적**: PT샵/헬스장 대상 Roomfit 도입 문의 유도
> **마지막 수정**: 2025-01-06

---

## 페이지 개요

### 전체 구조

```
1. Hero              - B2B 핵심 가치 (AI 카운팅, 코칭 집중)
2. Features          - 주요 기능 4가지
3. VBTAnalysis       - 전문 VBT 분석
4. TrainingModes     - 트레이닝 모드 (PT 맥락)
5. ROMOptimization   - ROM 개인화
6. MemberManagement  - 회원 관리 & 성과 분석
7. QuickStart        - 빠른 수업 진행
8. DeviceManagement  - 기기 관리
9. Marketing         - 마케팅 활용
10. ContactCTA       - 도입 문의
```

### 타겟 사용자

- **PT 트레이너**: 회원 관리 효율화, PT 차별화를 원하는 트레이너
- **헬스장/PT샵 운영자**: 시설 차별화, 회원 만족도 향상을 원하는 운영자
- **스포츠 센터**: 전문 VBT 장비 도입을 고려하는 시설

### 핵심 메시지

```
"카운팅은 AI에게, 트레이너는 코칭에 집중"
```

### 사용자 여정

```
[인지] Hero → [관심] Features → [전문성] VBT/Modes/ROM
→ [효율] Member/Quick → [신뢰] Device → [행동] Contact
```
