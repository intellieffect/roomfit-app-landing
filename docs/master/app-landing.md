# 앱 랜딩 페이지 마스터 문서

> **페이지**: 앱 (개인 사용자)
> **URL**: `/app`
> **목적**: Roomfit 앱의 AI/VBT 기능 소개 및 다운로드 전환
> **마지막 수정**: 2025-01-06

---

## 페이지 개요

### 전체 구조

```
1. Hero            - 앱 핵심 가치 (AI 분석)
2. Features        - 주요 기능 4가지
3. VBTAnalysis     - VBT 상세 설명
4. TrainingModes   - 트레이닝 모드
5. HandsfreeControl - 핸즈프리 조작
6. DataInsights    - 기록 & 성장 분석
7. Connectivity    - 연결성
8. DownloadCTA     - 앱 다운로드
```

### 타겟 사용자

- **개인 홈짐 사용자**: 트레이너 없이 혼자 운동하는 사람
- **데이터 중심 운동러**: 운동 기록과 성장을 추적하고 싶은 사람
- **효율 추구형**: 짧은 시간에 효과적인 운동을 원하는 사람

### 사용자 여정

```
[인지] Hero → [관심] Features → [이해] VBT/Modes
→ [편의성] Handsfree/Data → [행동] Download
```

| 4 | Zap | 1RM 안전 추정 | 위험한 테스트 없이 내 최대 근력을 파악할 수 있습니다 |

| **핵심 메시지** | 터치 한 번으로 바뀌는 운동의 차원 |
| **타겟 감정** | 편리함, 전문성 |

| 요소 | 텍스트 |
|------|--------|
| 뱃지 | 스마트 트레이닝 모드 |

| 제목 (line2) | 운동의 차원 |
| 부제 | 핀 교체 없이 터치 한 번으로 무게 설정을 끝내세요. |
| 설명 | 특히 드랍세트 설정 시, 플레이트 교체 없이 쉬지 않고 연속 수행이 가능합니다. 내 무게 수준에 맞게 비율(%)로 자동 계산됩니다. |

#### 트레이닝 모드

| # | 제목 | 설명 |
|---|------|------|

| 2 | Negative 모드 | 내리는 동작에서 더 강한 자극 → 근비대 효과 극대화 |
| 3 | Band 모드 | 관절에 부담 적은 초반 + 강한 마무리로 안전한 운동 |
| 4 | 자동 드랍세트 | "5kg씩 줄여가며 3단계" 단순 설정으로 완주 유도 |

### 기획

| 항목 | 내용 |

| **핵심 메시지** | 손을 놓지 않고, 말 한마디로 무게 조절 |
| **타겟 감정** | 편리함, 안전감 |

### 콘텐츠

| 요소 | 텍스트 |
|------|--------|
| 뱃지 | 핸즈프리 간편 조작 |
| 제목 (line1) | 손을 놓지 않고, |
| 제목 (line2) | 말 한마디로 무게 조절 |
| 부제 | "웨이트 온" 한 마디면 충분합니다. |
| 설명 | 고중량 운동 중 스마트폰을 조작하려고 자세를 풀지 마세요. 덤벨 내려놓고 핀 빼고 할 필요 없이, 음성 명령과 제스처만으로 운동 흐름을 유지할 수 있습니다. |

#### 음성 명령

```
"웨이트 온"
```

#### 제스처 기능

| # | 아이콘 | 제목 | 설명 |
|---|--------|------|------|
| 1 | ShieldAlert | Drop 자동 무게 Off | 그립을 놓치면 자동으로 무게 OFF로 안전 확보 |
| 2 | Pause | 휴식 자동 무게 Off | 가만히 있으면 자동으로 무게 OFF |
| 3 | Zap | 빠르게 당겨서 무게 On | 양쪽을 동시에 빠르게 당기면 무게 On |
