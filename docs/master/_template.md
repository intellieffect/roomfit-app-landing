## 작성 가이드

### 기획 섹션 작성 팁
- **목적**: "왜 이 섹션이 필요한가?"에 답하기
- **전환 목표**: 스크롤 유지, 클릭, 감정 변화 등 구체적으로
- **핵심 메시지**: 30자 이내로 압축

### 콘텐츠 작성 규칙
- 제목: 2줄 이내, 강조 키워드 분리
- 설명: 3문장 이내
- CTA: 행동 동사로 시작 (구매하기, 체험하기 등)

### 이미지 규칙
- 파일명: `섹션명-용도.확장자` (예: `hero-background.webp`)
- 권장 포맷: WebP (폴백으로 PNG)
- alt 텍스트: 이미지 내용을 설명하는 완전한 문장
