# Roomfit 앱 기능 명세서

## 개요

이 문서는 Roomfit 앱의 전체 기능을 정리한 명세서입니다. 각 기능별로 홈트레이닝 일반인 사용자와 PT샵/헬스장 B2B 타겟에 맞는 가치 제안을 포함합니다.

---

## 1. AI 퍼스널 트레이너 (VBT)

### 1.1 자동 음성 렙 카운팅

| 기능 | 설명 | 홈트 일반인 타겟팅 | PT샵/헬스장 타겟팅 |
| --- | --- | --- | --- |
| 자동 카운트 | 가동범위 기반 반복 횟수 자동 측정 | 혼자 운동해도 세는 데 신경 쓸 필요 없이 운동에만 집중 가능 | 트레이너가 카운팅 대신 자세 교정과 동기부여에 집중 가능 |
| 터닝 포인트 감지 | 반복의 시작/끝 지점 정확히 인식 | 동작을 끝까지 제대로 했는지 AI가 체크해줌 | 회원의 가동범위 일관성을 객관적으로 모니터링 |
| 렙 검증 | 유효 반복 카운트 (불완전 동작 필터링) | 대충 한 동작은 세지 않아서 스스로 정확한 운동 습관 형성 | 회원이 치팅 없이 정확히 수행했는지 데이터로 확인 가능 |

### 1.2 퍼포먼스 시각화

| 기능 | 설명 | 홈트 일반인 타겟팅 | PT샵/헬스장 타겟팅 |
| --- | --- | --- | --- |
| 실시간 라인 차트 | 속도/파워 변화를 실시간 그래프로 표시 | 운동 중 내 컨디션을 눈으로 확인하며 페이스 조절 | 회원에게 실시간으로 "지금 속도 떨어지고 있어요" 피드백 제공 |
| 세트별 비교 차트 | 세트 간 퍼포먼스 비교 시각화 | 오늘 몇 세트째에 지쳤는지 패턴 파악 | 회원별 세트 간 피로 패턴 분석하여 맞춤 세트 수 설계 |
| 히스토리 추이 | 과거 운동 기록과의 비교 분석 | "한 달 전보다 확실히 강해졌네!" 성장 실감으로 동기부여 | 회원에게 장기 성장 그래프 보여주며 PT 효과 입증 |

### 1.3 동작 분석

| 기능 | 설명 | 홈트 일반인 타겟팅 | PT샵/헬스장 타겟팅 |
| --- | --- | --- | --- |
| 동작 품질 평가 | 각 렙의 움직임 일관성 및 품질 점수화 | 트레이너 없이도 내 동작이 일정한지 점수로 확인 | 회원 동작 품질을 수치화하여 개선 포인트 명확히 제시 |
| 구심성 구간 (Concentric) | 수축 구간 속도/시간 분석 | (고급 사용자) 폭발적인 힘 쓰는 구간 개선에 활용 | 파워/근력 목표 회원에게 수축 속도 향상 코칭 근거 |
| 편심성 구간 (Eccentric) | 이완 구간 속도/시간 분석 | (고급 사용자) 천천히 내리는 훈련 제대로 하는지 확인 | 근비대 목표 회원에게 "더 천천히 내리세요" 데이터 기반 코칭 |
| 동작 패턴 인식 | 일관된 동작 패턴 여부 확인 | 피로해도 폼이 흐트러지지 않는지 AI가 감시 | 회원의 폼 무너짐 시점을 파악하여 세트 종료 타이밍 결정 |
| 운동 효율 지표 | 전체 세트의 효율성 종합 평가 | "오늘 운동 효율 85점" 같은 종합 피드백으로 만족감 | 회원별 운동 효율 리포트로 PT 성과 객관화 |

### 1.4 VBT 분석을 통한 전문 코칭

| 기능 | 설명 | 홈트 일반인 타겟팅 | PT샵/헬스장 타겟팅 |
| --- | --- | --- | --- |
| 속도 측정 | 각 렙의 평균/최대 속도(m/s) 실시간 측정 | 내가 얼마나 폭발적으로 들어올리는지 수치로 확인 | 선수/고급 회원에게 속도 기반 훈련 프로그램 설계 |
| 파워 계산 | 속도와 무게 기반 출력 파워(W) 산출 | "오늘 파워 신기록!" 새로운 성취 지표로 흥미 유발 | 운동선수 회원에게 파워 향상 추이 보고 및 훈련 조정 |
| 가속도 분석 | 동작 중 가속도 패턴 분석 | (고급 사용자) 초반 가속이 약한지 등 세부 개선점 파악 | 스포츠 특화 훈련 시 가속 패턴 최적화 코칭 |
| 피로도 감지 | 속도 저하 패턴으로 피로 상태 자동 감지 | "지금 피로 누적됨" 알림으로 부상 위험 전 운동 중단 유도 | 회원 과훈련 방지, 적정 볼륨에서 세트 종료 권고 |
| 1RM 추정 | 현재 속도 데이터 기반 1회 최대 중량 자동 계산 | 위험하게 1RM 테스트 안 해도 내 최대 근력 파악 가능 | 회원 1RM을 안전하게 추정하여 %1RM 기반 프로그램 설계 |
| RIR 예측 | 남은 반복 여력(Reps in Reserve) 실시간 표시 | 트레이너 없이도 "2개 더 할 수 있어요" AI가 알려줌 | 회원에게 최적의 Target Rep을 전문 데이터와 함께 코칭 |
| 속도 손실률 | 세트 내 속도 저하 비율(%) 모니터링 | 언제 세트를 끝내야 하는지 객관적 기준 제공 | 훈련 목표별(근력/근비대) 최적 속도 손실률로 세트 관리 |
| 상대 강도 | 현재 무게가 추정 1RM의 몇 %인지 표시 | "지금 내 최대의 75%로 운동 중" 강도 감각 형성 | 회원에게 "오늘은 80%로 5x5 갑니다" 전문적 프로그래밍 |
| 속도 구간 안내 | 훈련 목표별 권장 속도 범위 제시 | "근비대 원하면 이 속도 구간 유지하세요" 목표별 가이드 | 회원 목표(근력/파워/근비대)에 맞는 속도 존 설정 및 모니터링 |

### 3.1 빠르고 직관적인 운동 시작

| 휴식 시간 기록 | 세트 간 휴식 시간 추적 | 쉬는 시간도 기록되어 총 운동 패턴 파악 | 회원별 적정 휴식 시간 패턴 분석 및 조절 |
